## Usage

Import styles

File: `App.js`

```js
import '@rekamarket/set-ui/package/index.gen.css';
import { H1 } from '@rekamarket/set-ui';

<H1>Hello</H1>;
```

## Getting Started

Install dependencies,

```bash
$ yarn install
```

Compile rescript,

```bash
$ yarn rescript
```

Build documentation,

```bash
$ yarn dumi
```

Publish documentation via Github Pages,

```bash
$ yarn dumi:publish
```

## Interactive Playground

```bash
$ yarn install && yarn playroom
```
